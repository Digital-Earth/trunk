WorldView Studio System Requirements

Software

  • Operating System: Windows Vista, 7, or 8+
  • Web browser: Modern web browsers like the latest versions of Chrome, Firefox and Internet Explorer.

Hardware

  • Graphics card: 512MB and support for openGL 2.0
  • Disk Space: 100MB for installed application, and another 2GB for cached data
  • Memory: 2GB (4GB recommended)
  • Processor: Any CPU will do (multi-core processors recommended)
  • Internet connection speed: 10Mbps
