PYXIS GDAL Driver (RGB)

The PYXIS GDAL driver provides access to PYXIS from within a GDAL implementation. The driver loads a PYXIS PIPE file and provides access to the data in RGB format.

The driver input is a text file with extension “.pgd”. The contents of the file are as follows buy line number:
1. A folder where PYXIS will store its working files such as its cache.
2. A folder containing the PYXIS library files and the PYXIS plugins.
3. A PYXIS PIPE file.
4. RasterXSize: The number of pixels in the X dimension (integer).
5. RasterYSize: The number of pixels in the Y dimension (integer).
6. PYXIS resolution (integer).
7. Upper Left Longitude (double).
8. Upper Left Latitude (double).
9. Lower Right Longitude (double).
10. Lower Right Latitude (double).

The Well Known Text for the projection is: GEOGCS["GCS_WGS_1984",DATUM["D_WGS_1984",SPHEROID["WGS_1984",6378137,298.257223563]],PRIMEM["Greenwich",0],UNIT["Degree",0.0174532925199433]]

The GeoTransform is as follows:
adfGeoTransform[0]= Upper Left Longitude
adfGeoTransform[1]= ABS((Upper Right Longitude - Upper Left Longitude)/ RasterXSize)
adfGeoTransform[2]= 0
adfGeoTransform[3]= Upper Left Latitude
adfGeoTransform[4]= 0
adfGeoTransform[5]= ABS ((Lower Right Latitude - Upper Left Latitude)/RasterYSize);

There are three RasterBands: Red, Green and Blue. The data type is GDT_Int32, BlockXSize is RasterXSize and BlockYSize is 1.

See GDAL Tutorial for usage.
